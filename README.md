# Academex – School Management System

Academex is a full-stack school management system built from scratch using Node.js, MySQL, and vanilla JavaScript. It provides role-based access and core academic management features without relying on modern frontend frameworks.

---

## 🚀 Overview

Academex was built to manage student and administrative workflows in a structured way. The system focuses on backend fundamentals, database design, and implementing frontend behavior manually without frameworks.

---

## ✨ Key Features

- 👥 Role-based system (Admin / Users)  
- 📊 Student and academic data management  
- 🔐 Custom authentication and authorization system  
- ⚙️ Backend APIs for CRUD operations  
- 🌐 SPA-like frontend behavior without frameworks  

---

## 🛠 Tech Stack

**Frontend:**
- HTML, CSS  
- Vanilla JavaScript (DOM manipulation)  

**Backend:**
- Node.js  
- Express.js  

**Database:**
- MySQL

**Deployment**
- Vercel (Frontend)
- Railway (Backend + Database)
---

## 🧠 System Design Highlights

### 🔹 SPA-like Behavior (Without Frameworks)
- Implemented dynamic page updates using pure DOM manipulation  
- Avoided full page reloads by rendering content conditionally  
- Created a smooth user experience similar to Single Page Applications  

---

### 🔹 Custom Authentication System
- Designed a token-based authentication mechanism  
- Tokens generated on login and used for session validation  
- Implemented protected routes for role-based access control  
- Managed authentication flow without external auth libraries  

---

### 🔹 Database Design
- Designed relational schema for students, roles, and system data  
- Ensured efficient data handling using structured queries  
- Implemented full CRUD operations via backend APIs
